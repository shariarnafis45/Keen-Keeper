# 🚀 KeenKeeper — Keep Your Friendships Alive

## 📌 Overview

**KeenKeeper** is a modern friendship management web application designed to help users maintain meaningful relationships. It allows you to track interactions, set communication goals, and visualize your friendship activity over time.

This project focuses on improving real-life connections through simple tracking, reminders, and analytics — all wrapped in a clean, responsive UI.

## 🛠️ Technologies Used

* ⚛️ React.js / Next.js (App Router)
* 🎨 Tailwind CSS
* 📊 Recharts (for analytics visualization)
* 🔔 React Hot Toast (for notifications)
* 🌐 JSON (Local Data Source)
* 🔗 React Icons

---

## ✨ Key Features

### 1. 📋 Smart Friend Tracking

* View all friends with detailed profiles
* Track "days since last contact"
* Status indicators (Overdue / Almost Due / On Track)

### 2. ⚡ Quick Interaction Logging

* Instantly log **Call, Text, or Video** interactions
* Automatically updates timeline with date & activity
* Real-time toast notifications for user feedback

### 3. 📊 Friendship Analytics Dashboard

* Interactive Pie Chart showing communication patterns
* Visual breakdown of Call / Text / Video interactions
